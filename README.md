# Kairo AI Energy Advisor

Kairo AI Energy Advisor is a simple AI-driven tool designed to recommend solar energy solutions based on user needs.

## Features
- Estimates energy requirements
- Suggests suitable solar system types
- Provides cost ranges for different setups
- Offers simple, human-friendly energy insights

## Purpose
This project is part of AmunRa, an initiative focused on renewable energy, sustainability, and community impact. The goal is to explore how AI can support real-world decision-making in energy access.
